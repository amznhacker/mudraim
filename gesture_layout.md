# Mudra Keyboard Layout

## Hand Position Grid (3x3 per layer)

### Layer 1: Letters
```
Position (x,y):
(0,0) a  (0,1) b  (0,2) c
(1,0) d  (1,1) e  (1,2) f  
(2,0) g  (2,1) h  (2,2) i
(3,0) j  (3,1) k  (3,2) l
(4,0) m  (4,1) n  (4,2) o
(5,0) p  (5,1) q  (5,2) r
(6,0) s  (6,1) t  (6,2) u
(7,0) v  (7,1) w  (7,2) x
(8,0) y  (8,1) z  (8,2) space
```

### Layer 2: Numbers
```
(0,0) 1  (0,1) 2  (0,2) 3
(1,0) 4  (1,1) 5  (1,2) 6
(2,0) 7  (2,1) 8  (2,2) 9
(3,0) 0  (3,1) .  (3,2) ,
```

### Layer 3: Symbols
```
(0,0) !  (0,1) @  (0,2) #
(1,0) $  (1,1) %  (1,2) ^
(2,0) &  (2,1) *  (2,2) (
(3,0) )  (3,1) -  (3,2) +
(4,0) =  (4,1) [  (4,2) ]
(5,0) {  (5,1) }  (5,2) \
(6,0) |  (6,1) ;  (6,2) :
(7,0) "  (7,1) '  (7,2) <
(8,0) >  (8,1) ?  (8,2) /
```

## Gesture Commands

- **Tap**: Type character at position
- **Reverse Tap**: Backspace/Delete
- **Twist**: Switch between layers (Letters → Numbers → Symbols → Letters)
- **Double Twist**: Enter/Return
- **Pinch & Hold**: Modifier keys (Shift/Ctrl/Alt)
- **Reverse Pinch & Slide**: Arrow keys/Navigation

## Usage Example

To type "Hello!":
1. Tap (2,1) = h
2. Tap (1,1) = e  
3. Tap (3,2) = l
4. Tap (3,2) = l
5. Tap (4,2) = o
6. Twist (switch to symbols)
7. Tap (0,0) = !
